## Installation

### 1. Install dependencies (Arch Linux)

```bash
git clone https://github.com/TinaGrim/POS_accessories_CLI.git
cd POS_accessories_CLI
yay -S ncurses qrencode
```

### 2. Run Program

```bash
make run && make clean
```
